You can make custom directives in AngularJS too with app.directive.

New directives are created by using the .directive function. To invoke the new directive, make an HTML element with the same tag name as the new directive. When naming a directive, you must use a camel case name, w3TestDirective, but when invoking it, you must use - separated name, w3-test-directive:

You can invoke a directive by using:

Element name (i.e. <directive-name></directive-name>)

Attribute (i.e. <p directive-name></p>)

Class (i.e. <p class='directive-name'></p>)

Comment: you can add the directive name in a comment line in your html file, but you have to add the replace property in this example, otherwise the comment will be invisible. Note: You must add the value "M" to the restrict property to be able to invoke the directive from a comment.
